## INSTALLATION
Download the Project and install into:

C:\Program Files (x86)\Steam\steamapps\common\GarrysMod\garrysmod\

## USAGE

Start by simply wiring the inputs / outputs you want

Edit the config

Refresh the chip

Then let the E2 do the work for you
